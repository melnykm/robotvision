# Robot Harvester Vision System 🤖

## Description

This project is a computer vision system for a robot harvester. It classifies agricultural objects (such as fruits and other items) using machine learning algorithms.

The system is implemented in Python and includes:

* Perceptron (single-layer neural network)
* MLP (multi-layer perceptron with backpropagation)

## Features

* Training neural networks on synthetic data
* Visualization of loss and accuracy
* Comparison of Perceptron and MLP
* Confusion matrix and F1-score analysis
* Interactive testing using sliders

## Technologies

* Python 3
* NumPy
* Matplotlib
* Tkinter

## How to Run

1. Install dependencies:

```
pip install numpy matplotlib
```

2. Run the program:

```
python main.py
```
